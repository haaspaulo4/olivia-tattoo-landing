Como usar este kit

  • Todos os quadros estão em escala reduzida (zoom) só para visualização — o HTML interno já está no tamanho real do Instagram (1080×1350, 1080×1080, 1080×1920).
  • Para exportar: abra o arquivo, use "Inspecionar elemento" → screenshot do nó, ou copie o bloco `.slide` / `.feedpost` / `.story` / `.reels` para uma ferramenta como Figma/Canva mantendo as mesmas cores e fontes.
  • Fontes usadas: Cormorant Garamond (títulos, itálico fino — remete ao traço da agulha), Jost (texto/legendas) e Caveat (assinatura "olivia", uso pontual).
  • Paleta: fundo tinta #1B1613, papel #F3EDE4, traço dourado #C9A671 — evita o clichê "cream + terracota" e conecta com tinta/pele/ouro fino.
  • O elemento de assinatura é a moldura de linha fina (1px, opacidade baixa) que aparece em todo quadro — reforço visual do "traço contínuo" da marca.
